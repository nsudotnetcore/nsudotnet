Задача о ранце

Необходимо реализовать веб-приложение, которое может решить задачу о ранце. Алгоритм решения не важен, возможен прямой перебор

UI ПРИЛОЖЕНИЯ

UI приложения должен состоять из следующих страниц:

Описание

1) Страница «Список выполняющихся задач», содержащая столбцы:
  1. Наименование задачи (ссылка на переход к странице деталей в статусе «Выполнено»)
  2. Вместимость рюкзака
  3. Максимальная ценность (если задача в статусе «Выполнено»)
  4. Процент выполнения (если задача в статусе «Выполняется»)
  5. Статус «Выполняется»/ «Выполнено»
  6. Действие «Удалить» (должно прерывать выполняющуюся задачу)
  7. Действие «Добавить новую задачу»
2) Страница с формой создания новой задачи:
  1. Наименование задачи
  2. Вместимость рюкзака
  3. Таблица с наименованием, ценностью и весом предметов (для ввода)
3) Страница, с деталями задачи, отображающая:
  1. Условие
    1. Наименование задачи
    2. Вместимость рюкзака
    3. Таблица с наименованием, ценностью и весом предметов (для чтения)
  2. Решение
    1. Максимальная ценность
    2. Таблица распределения предметов для достижения оптимальной ценности
    3. Время выполнения

Важно

1) Необходимо запускать вычисление нескольких задач одновременно

2) Решение должно отображать прогресс выполнения задания в процентах

3) Вычисление должно уметь «переживать» запуск веб-сервера, под которым оно работает, т.е. при перезапуске процесс должен продолжаться с того места где он остановился

Технические условия

1) .NET Core

2) Решение должно быть реализовано либо с использованием ASP.NET MVC, либо в виде SPA, с использованием одной их технологий: Razor, Angular 7, Vue, либо React
