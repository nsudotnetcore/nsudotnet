Система оценок

Задания можно выбирать любые и делать их в любом порядке.

- На «удовлетворительно» - достаточно 8 баллов

- На «хорошо» - достаточно 12 баллов

- На «отлично» - достаточно 15 баллов

Что нужно?

Для выполнения заданий вам понадобится:

Visual Studio 2017 (Community) или Visual Studio Code

MS SQL Server Express

При выполнении задач приоритет лучше отдавать .NET Core. Для последних задач он - обязательное условие.
